# Security

If you believe you have found a security vulnerability in **telephony_context**, please report it responsibly:

1. **Do not** open a public GitHub issue for undisclosed vulnerabilities.
2. Use [GitHub Security Advisories](https://github.com/6litz/telephony-context/security/advisories/new) for this repository (if enabled), or contact the maintainers through a private channel they provide.

Include enough detail to reproduce the issue (Flutter/Android versions, plugin version, and a minimal description).

This plugin is intentionally limited to non-sensitive Telephony APIs and does not collect IMEI, IMSI, ICCID, or phone numbers; still, reports about unexpected data exposure or crashes are welcome.
